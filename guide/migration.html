# Migration

# v1

@random-di/container is a rewrite @botflx/dependency-injection-container. You can migrate from the first package by using @random-di/v1-loader (opens new window).

# v2@beta

@random-di/container was originally hosted in @botflx/dependency-injection-container@beta so you can just run:

npm uninstall @botflx/dependency-injection-container
npm install @random-di/container
